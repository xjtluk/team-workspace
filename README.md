# 团队工作室 (Team Workspace)

像素风格团队协作工作室 — 小马、CC、KK 的线上协作空间。

## 技术栈

- **后端**：Node.js + Express + WebSocket + SQLite
- **前端**：Preact + Canvas 2D 像素网格渲染
- **角色**：CSS 像素方块手绘（非贴图），支持独立动画与状态切换

## 项目结构

```
team-workspace/
├── docs/           ← 项目文档（PRD / 技术方案 / 原型）
├── assets/         ← 静态资源（角色素材、图标）
├── src/            ← 源代码（待 CC 开工）
└── .gitignore
```
